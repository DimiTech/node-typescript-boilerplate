# node-typescript-boilerplate

A starting point for Node.js projects written in TypeScript.

## Run it:

```
make
make start
```

## What this boilerplate includes?

```
- Node.js environment
- ES6 Modules
- Dependency Injection
- Source Map Support
- Absolute path imports
- Watcher
- Linting
```
